# Joke API
This is a simple Spring Boot app that provides random jokes through a RESTful API.

## Getting Started
To use the Joke API, follow these steps:

### Clone the repository to your local machine.
### Build the project using Maven:
           mvn clean install

### Start the app
            mvn spring-boot:run
